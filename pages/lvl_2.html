Level 2: Broken Authentication

What is Broken Authentication?

Broken authentication is gaining access to a user's account by inputting
the most common passwords. This is usually done via a script, where the
script will be given a list of the most common passwords and loop through
each password, attempting to gain access to an account. This is what you
will be learning to do in this exercise.

Exercise:

The goal of this exercise is to write a script that will loop through
passwords in a file and attempts to input each password until you gain
access to an account. Since there is a limited number of common passwords
and you should be writing a script for this exercise, we have generated a
password file containing 100,000 passwords with randomly selected letters.
You are still welcome to input each password individually, but we highly
recommend writing a script as it will save you hours of time.
